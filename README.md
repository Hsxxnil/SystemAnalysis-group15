# 電商訂貨系統

## 範疇
1. 電商透過拍賣平台取得消費者的訂單，將每日產品需求統整並填入此訂貨系統，且填單者填完後由另一位同事審核，確認無誤後，向上游廠商進行訂購。
2. 系統每月依據使用者回報的訂單完成狀況，自動產生報表讓使用者與上游廠商進行核對，確認無誤再進行付款。
## 目的
減少電商倉儲成本，達成及時訂購、出貨。
